# Requirements

In order to test this design on hardware, you will need the following:

* Vivado 2025.2
* Vitis 2025.2 (for the standalone lwIP echo-server flow)
* PetaLinux Tools 2025.2
* [Ethernet FMC Max]
* One of the supported carrier boards listed below
